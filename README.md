**Analisis Review E-Commerce Berdasarkan Tipe Informasi Menggunakan IndoBERT**

Repository ini berisi implementasi kode program untuk penelitian tugas akhir
mengenai klasifikasi aspek ulasan e-commerce berbahasa Indonesia menggunakan
tiga pendekatan, yaitu TF-IDF + Support Vector Machine (SVM), Bidirectional
Long Short-Term Memory (BiLSTM), dan IndoBERT.
